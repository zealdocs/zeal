Usage

After installing/compiling, to quickly download documentation sets, click 'Edit', 'Options', 'Docsets', 'Download...'.

Docset filter

You can filter the search results to a specific (or a set of specific) docsets.

The docset filter and the search query are delimited by a colon character (:).

Examples:

  • ruby:set_x will search all docsets containing ruby (e.g. ironruby, ruby and ruby2) for set_x
  • set_x will search all docsets for set_x

Editor plugins

Emacs package

There is an Emacs package available at https://github.com/jinzhu/zeal-at-point.

Sublime Text package

There is a Sublime Text 2/3 package available at https://github.com/vaanwd/Zeal.

Vim integration

There is a Vim plugin available at https://github.com/KabbAmine/zeavim.vim.

Or you can use https://github.com/Keithbsmiley/investigate.vim with something like this in .vimrc:

let g:investigate_command_for_python = '/usr/bin/zeal --query ^s'

A simpler way to integrate with Vim is a simple mapping, like

:nnoremap gz :!zeal --query "<cword>"&<CR><CR>

Brackets integration

There is a Brackets integration available at https://github.com/anephew/brackets-zeal.

IntelliJ IDEA integration

There is an IDEA plugin available at https://plugins.jetbrains.com/plugin/7351?pr=idea.
